вспоминаем старый материал
1) три правила Python
 - правило порядка (код выполняется сверху вниз кроме ленивых вычислений)
 - синтаксис языка (написание кода)
 - правило начала (код не может начинаться с пробела)
2) основные кострукции Python
 - системные (встроенные) функции (print, input, len, int, str и т.д.)
 - переменные (в Python ссылочная система переменных)
 - условный операторы (if, elif, else)
 - циклы (прямой цикл while, цикл-итератор for)
 - функции и модули (есть определение, есть вызов (не забываем про ленивые вычисления))
 - введение в ООП (класс, объект, свойства, методы)
 - структура данных
 - практика (игры, консольные приложенияб оконные приложения)
3) парадигма объекто-ориентированного программирования
объект - сущность, имеющая свойства и методы
класс - заготовка для объектов

можно программировать взаимодействия объектов между собой
